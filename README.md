# Personal Library Manager

A library management app built with Python and Kivy.

## Features

- Add, remove, search books
- Borrow and return books
- View library statistics
- Auto-save to JSON

## How to Run

1. Install Kivy:
   pip install kivy

2. Run the app:
   python main.py

## Project Structure

- main.py - Entry point
- View.py - User interface
- Controller/ - Logic
- Model/ - Data classes
- data/ - JSON storage
